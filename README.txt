Учебный проект для TIZEN SDK 2.2.1
Рассмотрен процесс поэтапного создания простого
нативного приложения для ведения списков покупок.

Список изменений:
1) В IDL_PANEL1 добавлена ListView (через UI Builder).
2) IDC_LISTVIEW1 и IDC_LABEL1 поменяны местами с помощью текстового редактора. 
3) Размеры ListView привязаны к размерам IDL_PANEL1.
